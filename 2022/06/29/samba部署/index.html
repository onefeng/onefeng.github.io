samba部署

2022/06/29 samba 共 477 字，约 2 分钟

samba主要用户windows与linux的文件共享，这里说明samba的部署

1.安装

sudo apt-get install samba samba-common

2.添加用户

adduser share
passwd share

添加samba用户

sudo smbpasswd -a share

3.设置共享的目录权限

mkdir sm
sudo chmod 777 /home/share/sm

4.配置samba

sudo vim /etc/samba/smb.conf

在最后面添加配置

[share]
comment = share folder
browseable = yes
path = /home/share/sm
create mask = 0755
directory mask = 0755
valid users = share
force user = share
force group = share
public = yes
available = yes
writable = yes

5.重启samba

sudo /etc/init.d/smbd restart

6.使用

在windows下 win+R 输入 \hostname\share 打开 输入用户密码
